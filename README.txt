README - Raj Sen Portfolio Website (package)
-------------------------------------------
Files:
  - index.html
  - styles.css
  - scripts.js
  - README.txt
  - assets/thumbs/ (place your thumbnail images here, name like thumb01.jpg ... thumb12.jpg)

How to use:
  1. Unzip the package.
  2. Replace images in assets/thumbs/ with your actual thumbnails (keep filenames or edit index.html).
  3. Open index.html in browser to preview.
  4. All styles are in styles.css and interactions in scripts.js (separate files as requested).
  5. To host: upload files to any static host (Netlify, Vercel, GitHub Pages).

Customization tips:
  - Change your email in scripts.js contact button (mailto).
  - Edit hero text and services.
  - To add more thumbnails, copy one <figure class='thumb'> block in index.html and adjust data-src / img src.
  - For animated exported MP4/GIF previews, link them in data-src (modal supports direct links).
